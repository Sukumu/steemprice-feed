# Steem-Price-Feed

Update Steem Price Feed for Witness using CoinGecko API and WebSocket.

## Prerequisite

- Node.js (v8+)
- Makefile installed

## Setup

- git clone this repo
- run `make init` to install pm2, skip this step if you already install pm2
- run `make start` (start the tool with pm2)
